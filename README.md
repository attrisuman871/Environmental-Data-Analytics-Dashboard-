# Ecosystem Health Analysis: Impact of Heavy Metal Contamination

## Overview

This project investigates how heavy metal contamination affects ecosystem health using environmental monitoring data from the **Firth of Clyde, Southwest Scotland**.

The analysis focuses on the relationship between **trace metal concentrations and ecological indicators such as species biomass and biodiversity**.

The goal is to determine whether heavy metal contamination can act as an indicator of ecosystem stress.

---

# Research Question

Are heavy metals affecting ecosystem health?

Specifically:

- Do high metal concentrations correspond to lower species biomass?
- Is species diversity negatively affected by contamination?
- Which sites show the strongest environmental stress signals?

---

# Dataset

The dataset contains environmental monitoring data collected from multiple sampling sites.

### Variables

Trace Metals:

- Cd (Cadmium)
- Co (Cobalt)
- Cr (Chromium)
- Cu (Copper)
- Mn (Manganese)
- Ni (Nickel)
- Pb (Lead)
- Zn (Zinc)

Ecological Indicators:

- Species Diversity
- Total Biomass

Sampling Sites:

```
S1 – S12
```

---

# Analytical Approach

The project applies several data analysis techniques:

### Descriptive Analysis
Summarizing contamination levels and biomass distribution across sampling sites.

### Diagnostic Analysis
Examining relationships between metal concentrations and ecological indicators.

### Correlation Analysis
Identifying statistical relationships between environmental parameters.

### Visual Analytics
Using dashboards to explore environmental patterns interactively.

# Key Insights

- Metal contamination varies significantly across sites.
- Sites with higher contamination often show reduced biomass.
- Strong correlations exist among several heavy metals.
- Species diversity tends to decrease in highly contaminated environments.
